# Decentralized Education / 去中心化教育平台

> 课程内容存储在 Walrus，AI Agent 根据学习进度个性化推荐，构建开放可信的教育生态

## 解决什么问题

在线教育平台中心化控制课程内容，创作者无法真正拥有自己的教学内容。学习者数据被平台垄断，跨平台学习进度无法互通。统一的教学大纲无法满足不同基础和学习风格的学员需求。本方案将课程内容存储在 Walrus 确保永久可访问，AI Agent 追踪学习进度并提供个性化学习路径，链上颁发可验证的结业证书。

## 核心功能

- 去中心化课程托管：课程视频、文档、习题等内容存储在 Walrus，创作者完全控制
- AI 个性化学习路径：Agent 根据学员知识水平、学习进度和目标定制学习计划
- 链上学分认证：完成课程后颁发链上可验证的学分证书，跨机构认可

## 使用的 Walrus / Sui 能力

| 能力 | 用途 |
|------|------|
| Walrus Blob 存储 | 存储课程视频、文档、习题、学习记录 |
| Sui 对象模型 | 链上记录课程注册、学分证书、学习进度 |
| Sui 代币机制 | 课程付费和创作者激励的链上结算 |
| Seal 加密 | 加密付费课程内容，仅注册学员可访问 |

## 实现方案

### Agent / AI 部分

- 水平评估 Agent 通过诊断测试评估学员初始知识水平
- 路径规划 Agent 根据学习目标和当前水平生成个性化学习路径
- 答疑 Agent 实时回答学员问题，结合课程内容给出针对性解答
- 进度追踪 Agent 监控学习进度，识别薄弱环节并调整学习计划
- 作业批改 Agent 自动批改编程作业和客观题，给出改进建议

### Walrus 集成

- 课程视频以 `courses/{course_id}/videos/{lesson_id}.mp4` 存储
- 课程文档以 `courses/{course_id}/docs/{doc_id}.pdf` 存储
- 习题库以 `courses/{course_id}/exercises/{exercise_id}.json` 存储
- 学习记录以 `learning/{user_id}/progress/{course_id}.json` 存储进度
- 证书和学分以 Sui 链上对象颁发，记录完成课程和成绩

### 前端 / 后端

- 前端：React 学习平台，支持视频播放、互动练习、进度追踪
- 后端：Python AI 教学服务 + Move 学分合约
- 关键页面：课程大厅页、学习界面页、进度仪表盘页、证书页

## 技术架构

创作者上传课程内容 → 内容写入 Walrus → Sui 链上注册课程 → 学员注册课程 → 水平评估 Agent 诊断 → 路径规划 Agent 生成学习计划 → 学员按计划学习 → 从 Walrus 加载课程内容 → 进度追踪 Agent 记录学习数据 → 答疑 Agent 实时辅助 → 作业批改 Agent 评估练习 → 完成课程后 → 链上颁发学分证书 → 学习记录持久化到 Walrus。
