# Decentralized Academic Publishing / 去中心化学术出版

> 论文存储在 Walrus，AI 辅助同行评审，引用关系上链可追溯，重构学术出版生态

## 解决什么问题

传统学术出版被少数出版商垄断，论文获取成本高昂，审稿周期漫长。审稿过程不透明，论文修改历史不可追溯。引用关系缺乏可靠的链上记录，难以准确评估学术影响力。本方案将论文和审稿记录存储在 Walrus，AI 辅助加速同行评审，引用关系上链建立可信的学术影响力指标，推动学术出版的开放和透明。

## 核心功能

- 去中心化论文存储：论文全文、附件、数据集存储在 Walrus，永久可访问且不可篡改
- AI 辅助评审：Agent 预审论文质量，检查方法学漏洞，为审稿人提供审稿辅助
- 链上引用追踪：论文引用关系记录在 Sui 链上，构建可验证的学术影响力图谱

## 使用的 Walrus / Sui 能力

| 能力 | 用途 |
|------|------|
| Walrus Blob 存储 | 存储论文全文、审稿记录、补充材料 |
| Sui 对象模型 | 链上记录论文注册、引用关系、影响力指标 |
| Sui 代币机制 | 论文发表费用和审稿激励的链上结算 |
| Seal 加密 | 加密双盲审稿中的作者身份信息 |

## 实现方案

### Agent / AI 部分

- 格式审查 Agent 检查论文格式、参考文献完整性、图片质量
- 方法学检查 Agent 分析实验设计和统计方法，识别潜在问题
- 查重 Agent 将论文与 Walrus 上的已有论文比对，检测抄袭和重复发表
- 综述生成 Agent 自动生成论文摘要和领域综述，辅助审稿人快速理解
- 引用追踪 Agent 自动识别论文引用关系，更新链上引用图谱

### Walrus 集成

- 论文全文以 `papers/{doi}/main.pdf` 存储在 Walrus
- 补充材料以 `papers/{doi}/supplements/{filename}` 存储
- 审稿记录以 `papers/{doi}/reviews/{review_id}.json.enc` 加密存储
- 修改历史以 `papers/{doi}/versions/v{N}.pdf` 存储每个版本
- 论文元信息和引用关系以 Sui 链上对象存储，支持查询和统计

### 前端 / 后端

- 前端：React 学术平台，支持论文提交、审稿、检索、引用分析
- 后端：Python AI 评审服务 + Move 论文注册合约
- 关键页面：论文提交页、审稿工作台、论文检索页、影响力分析页

## 技术架构

作者提交论文 → 格式审查 Agent 预检 → 查重 Agent 比对已有论文 → 论文写入 Walrus → Sui 链上注册 DOI → Seal 加密作者信息实现双盲 → 方法学检查 Agent 辅助审稿 → 审稿人提交审稿意见 → 审稿记录加密存入 Walrus → 作者修改提交新版本 → 通过后链上标记已发表 → 引用追踪 Agent 更新引用图谱。
