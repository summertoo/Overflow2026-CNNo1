# Decentralized Media Platform / 去中心化媒体平台

> 图片和视频存储在 Walrus，AI 自动生成标签和描述，链上管理版权和付费，构建创作者经济

## 解决什么问题

中心化媒体平台对创作者不公平，平台单方面控制内容分发和收益分配，创作者随时可能被删号封禁。媒体文件存储成本高昂，传统 CDN 分发效率受限。本方案利用 Walrus 的大容量低成本存储托管媒体文件，AI 自动化内容标签和描述生成，Sui 链上管理版权和付费，构建公平透明的去中心化创作者经济。

## 核心功能

- AI 内容标注：自动为上传的图片/视频生成标签、描述、分类，提升内容可发现性
- 链上版权管理：创作者上传时自动在链上注册版权，支持许可协议和付费墙设置
- 去中心化内容分发：通过 Walrus 直接分发内容，无中心化平台干预

## 使用的 Walrus / Sui 能力

| 能力 | 用途 |
|------|------|
| Walrus Blob 存储 | 存储图片、视频、缩略图等媒体文件 |
| Sui 对象模型 | 链上记录版权信息、许可协议、收益分配 |
| Sui 代币机制 | 实现内容付费和创作者收益自动分配 |
| Walrus 读取 API | 高效分发媒体内容给消费者 |

## 实现方案

### Agent / AI 部分

- 内容分析 Agent 上传时自动识别图片/视频内容，生成标签和描述
- 质量评估 Agent 评估内容质量、原创性和合规性
- 推荐 Agent 根据用户偏好和内容特征进行个性化推荐
- 版权保护 Agent 检测侵权内容，通过感知哈希比对已有作品
- 翻译 Agent 自动将内容描述翻译为多语言，扩大受众范围

### Walrus 集成

- 原始媒体以 `media/{creator_id}/{content_id}/original.mp4` 存储
- 缩略图以 `media/{creator_id}/{content_id}/thumb.jpg` 存储
- AI 标注以 `media/{creator_id}/{content_id}/annotations.json` 存储
- 多语言描述以 `media/{creator_id}/{content_id}/i18n/{lang}.json` 存储
- 版权和收益配置以 Sui 链上对象管理，包含许可协议和分润比例

### 前端 / 后端

- 前端：React 媒体平台，支持上传、浏览、付费、创作者仪表盘
- 后端：Node.js API 服务 + Move 合约
- 关键页面：首页推荐流、内容详情页、创作者工作室、钱包收益页

## 技术架构

创作者上传媒体文件 → 内容分析 Agent 生成标签描述 → 质量评估 Agent 审核内容 → 版权保护 Agent 检查侵权 → 媒体文件和标注写入 Walrus → Sui 链上注册版权 NFT → 翻译 Agent 生成多语言描述 → 推荐 Agent 索引内容 → 消费者浏览内容 → 从 Walrus 加载媒体 → 付费内容通过链上交易解锁 → 收益自动分配给创作者。
