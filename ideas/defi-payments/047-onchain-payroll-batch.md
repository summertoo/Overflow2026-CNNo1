# 工资链上化方案 / On-Chain Payroll Solution

> 企业通过 PTB 批量发放工资，支持多币种、多链、自动税务

## 解决什么问题

远程团队和 DAO 的工资发放面临多币种、多地区和合规要求复杂的挑战。财务人员需要手动处理汇率转换、税务预扣和跨链转账。工资链上化方案将整个发薪流程搬到链上，通过 PTB 批量执行多币种工资发放，自动处理兑换、税务和跨链转账，大幅提升发薪效率。

## 核心功能

- 批量多币种发薪：一次操作向多名员工发放不同币种的工资
- 自动税务预扣：根据员工所在地区自动计算和扣除税款
- 跨链工资发放：支持将工资跨链发送到员工在其他链的地址
- 工资流模式：支持将月薪转换为按日发放的流式支付
- 薪资模板管理：支持保存常用薪资配置模板，快速复用历史发薪方案
- 发薪记录审计：所有发薪操作链上留痕，支持导出合规报告

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| PTB | 批量执行多员工的"兑换 → 税务扣除 → 转账" |
| Coin 标准 | 处理多种工资代币和税务代币 |
| 对象所有权 | PayrollConfig 作为企业对象，管理员工和配置 |
| 动态字段 | 存储员工配置（币种偏好、税率、地址） |
| 共享对象 | 企业薪资池作为共享对象 |

## 实现方案

### 智能合约 (Move)
- `payroll_engine` 模块：核心发薪引擎，处理批量转账和兑换
- `tax_withholding` 模块：按地区税法自动计算和扣除税款
- `cross_chain_bridge` 模块：通过跨链桥将工资发送到其他链
- `streaming_payroll` 模块：将月薪转换为按日/周的流式支付
- 关键数据结构：`PayrollBatch { company: address, payments: vector<EmployeePayment>, total_amount: Coin<T>, tax_collected: Coin<T>, executed_at: u64 }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK
- 关键页面：Employee Management（员工管理）、Run Payroll（执行发薪）、Tax Dashboard（税务面板）、Payment History（发放历史）、Stream Manager（流式支付管理）
- 后端提供税率计算和跨链桥集成
- 支持通过 CSV 批量导入员工信息和薪资数据，简化大型团队配置

## 技术架构

管理员在后台配置员工信息（地址、薪资、目标币种、税务地区）。发薪日管理员触发批量发薪，系统构建大型 PTB：遍历每位员工 → 计算税后金额 → 通过 DEX 兑换为目标币种 → 扣除税款到税务池 → 转入员工地址。需要跨链的部分通过跨链桥模块处理。流式支付模式下，月薪被拆分为按日发放的流，通过流式支付合约自动执行。税务面板展示各地区税款汇总供企业报税。
