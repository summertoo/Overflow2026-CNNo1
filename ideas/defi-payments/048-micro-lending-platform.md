# 微型贷款平台 / Micro-Lending Platform

> 基于链上信用的微型贷款，自动审批、即时放款、智能还款

## 解决什么问题

全球数十亿人无法获得传统银行信贷服务，小额借款需求（几十到几百美元）在传统金融中因审核成本高而不被服务。现有 DeFi 借贷协议要求超额抵押，无法满足信用贷需求。微型贷款平台基于用户的链上行为数据（交易频率、资产持有、历史还款记录）建立信用评分，为信用良好的用户提供无抵押或低抵押的微型贷款，审批自动化、放款即时完成，降低金融服务门槛。

## 核心功能

- 链上信用评分：基于用户的链上行为（交易历史、资产规模、DeFi 参与度）自动计算信用分数
- 自动审批放款：信用达标的用户申请后合约自动审批，资金即时到账
- 智能还款管理：支持一次性还款和分期还款，逾期自动计算罚息
- 流动性提供者市场：LP 向资金池存入资产赚取贷款利息，利率由资金利用率动态调整
- 信用成就体系：按时还款的用户获得信用成就 NFT，解锁更高额度和更低利率

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | Loan 对象封装贷款状态，CreditScore 对象封装信用数据 |
| PTB | 原子化执行"信用查询 → 审批判定 → 资金发放" |
| Coin 标准 | 处理贷款发放、还款和利息的多币种操作 |
| 动态字段 | 存储还款计划、逾期记录和信用事件 |
| 共享对象 | 贷款池作为共享对象，支持多用户并发借贷 |

## 实现方案

### 智能合约 (Move)
- `credit_engine` 模块：基于链上数据计算信用评分，支持多维度加权
- `loan_factory` 模块：根据信用评分决定贷款额度、利率和抵押率
- `repayment_scheduler` 模块：管理分期还款计划、逾期检测和罚息计算
- `liquidity_pool` 模块：管理放贷资金池，支持 LP 存入赚取利息
- 关键数据结构：`Loan { borrower: address, principal: Coin<T>, interest_rate: u64, due_date: u64, status: LoanStatus, repayment_schedule: vector<Installment>, credit_score_used: u64 }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK + Recharts
- 关键页面：Loan Application（贷款申请）、My Loans（我的贷款）、Repayment Portal（还款门户）、Credit Dashboard（信用面板）、Liquidity Provider（流动性提供者页面）
- 后端索引链上事件计算信用评分并发送还款提醒
- 信用评分算法加权维度：交易频率（20%）、资产总量（20%）、DeFi 协议参与度（20%）、历史还款记录（30%）、钱包活跃时长（10%）

## 技术架构

用户连接钱包后，credit_engine 自动扫描其链上历史（交易次数、资产总量、DeFi 交互记录、过往贷款还款情况），计算 0-1000 的信用评分并写入 CreditScore 对象。用户申请贷款时，合约根据信用评分自动判定：信用分 > 阈值则通过审批，额度由信用分映射表决定。通过 PTB 从流动性池中提取资金转入借款人地址，同时创建 Loan 对象跟踪还款状态。还款时通过 PTB 将还款金额（本金 + 利息）转入流动性池，Loan 对象状态更新。LP 向流动性池存入资金赚取贷款利息收益，利率由资金利用率动态调整。
