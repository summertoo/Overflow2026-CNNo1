# Gamified Predict App / 游戏化预测应用

> 化身虚拟角色，预测正确获得经验和装备，排行榜和社交分享

## 解决什么问题

传统预测市场缺乏趣味性和社交吸引力，难以吸引年轻用户和游戏玩家。Gamified Predict App 将预测交易包装为 RPG 游戏——用户创建虚拟角色，每次成功预测获得经验和装备，角色等级与预测能力挂钩。通过游戏化机制提升用户粘性和活跃度，同时保持 Predict 合约的真实交易逻辑。

## 核心功能

- 虚拟角色系统：创建和培养预测者角色，等级反映预测能力
- 经验值和升级：成功预测获得 XP，累积升级解锁新能力
- 装备系统：连胜奖励装备道具，提升角色属性
- 社交排行榜：按等级和胜率排名，支持分享到社交媒体

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::mint | 预测交易的核心执行 |
| predict::redeem_permissionless | 结算后赎回并计算 XP |
| PredictManager | 管理用户仓位 |
| Sui NFT | 角色和装备作为链上 NFT |

## 实现方案

### 智能合约 (Move)
- `character_nft` 模块：角色创建和属性管理
- `equipment_nft` 模块：装备铸造和装备逻辑
- `xp_system` 模块：经验值记录和升级逻辑
- 关键数据结构：`Character`（等级、XP、装备槽位）、`Equipment`（属性加成、稀有度）

### 前端 / 后端
- 技术栈：React + Three.js / Lottie + Node.js
- 角色创建和自定义界面
- 预测战斗界面：选择预测方向和金额，动画展示结果
- 角色面板：属性、装备、等级进度
- 排行榜：全服排名和好友排名
- 社交分享：生成带角色信息的分享卡片

## 技术架构

用户创建角色 → 选择预测方向和金额 → predict::mint 执行交易 → 到期 redeem 计算结果 → 胜利获得 XP 和装备 → 链上更新角色 NFT 属性 → 排行榜刷新 → 可分享战果。
