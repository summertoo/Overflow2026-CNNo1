## Description / 项目简介

DeepPilot 是一个面向 DeepBook Predict 的 AI 辅助交易审查终端，把用户在 Web 或 Telegram 里的自然语言想法转成可检查的预测市场操作：先发现实时 BTC 市场、生成报价和风险说明，检查 oracle 新鲜度、流动性、余额、gas 和交易状态，同时也可以收集金融新闻，提供交易需要的参考信息，再把最终动作交给钱包确认。它重要的地方在于把“AI 帮我理解和组织交易”和“用户审查并签名”结合降低新手初次使用预测市场的操作门槛，同时避免黑箱自动下单，让每次执行都有上下文、风控和可追踪记录。
