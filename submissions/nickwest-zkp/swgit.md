## Description / 项目简介

swgit is a decentralized Git collaboration prototype built on Sui and Walrus. 
It combines Walrus for durable blob storage with Sui for identity, permissions, repository metadata, and branch coordination. 
Developers and AI agents can publish Git objects to Walrus, update repository refs through signed Sui transactions, and reconstruct full Git histories from on-chain repository state. 
The result is a practical, wallet-native workflow for programmable, verifiable, and agent-friendly source code collaboration.
