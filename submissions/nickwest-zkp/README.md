## Description / 项目简介

SuiBlobs is a full-stack Sui dApp workspace for wallet-bound AI NFT minting. It combines a quantized image generator,
a browser-based frontend, a Move smart contract, Walrus-hosted model weights, a trusted inference service, and a
Groth16 ZK proof flow so a wallet address can deterministically generate, verify, and mint a Blob NFT on-chain under
local, TEE, or ZK verification modes.
