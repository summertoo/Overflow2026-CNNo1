## Description / 项目简介

Strike5 is a short-cycle BTC prediction arena powered by DeepBook Predict on Sui. Users can connect a Sui wallet, open fixed-risk BTC Above / Below / Range positions, track live PnL, cash out before expiry, or hold to oracle settlement. Around the core DeepBook Predict lifecycle, Strike5 adds a lightweight arena layer with streak combos, opt-in leaderboard, market feed, verified position snapshots, and sealed-call experiments for a more consumer-friendly prediction market experience.
