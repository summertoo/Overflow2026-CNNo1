## Description / 项目简介

VolShape Studio is a payoff-first execution interface for DeepBook Predict. Instead of asking users to pick strikes and legs first, the product lets users draw the terminal payoff curve they want, compiles that curve into the closest executable DeepBook Predict strategy, routes execution through a connected Sui testnet wallet, and tracks resulting positions in Portfolio.

VolShape Studio 是一个面向 DeepBook Predict 的“收益曲线优先”交易前端。用户不需要先理解 strike、leg 和组合结构，而是直接画出希望在到期时获得的收益曲线；系统把这条曲线编译为最接近的可执行 Predict 策略，通过 Sui 测试网钱包完成执行，并在 Portfolio 中追踪仓位和赎回状态。
