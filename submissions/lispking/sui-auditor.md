## Description / 项目简介

本地优先的 Rust CLI 工具，用于审计 Sui Move 仓库。通过 OpenAI 兼容模型实现从仓库路径到安全审计报告的轻量工作流：发现包 → 健康检查 → 收集上下文 → 调用模型 → 输出 Markdown 报告。支持一键审计和交互式 REPL 两种模式，字节预算约束确保 prompt 大小可控。
