## Description / 项目简介

Sui 生态趋势聚合看板，用免费公开数据源替代付费社交 API，将"为什么现在在涨"转化为可审计、可解释、可追溯的信号集。当前版本聚焦快照式聚合 MVP：离线同步脚本抓取 Sui Blog、GitHub、GraphQL、DefiLlama 等 12 个公开数据源，生成趋势排名、项目矩阵和信号溯源页面。
