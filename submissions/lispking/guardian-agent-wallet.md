## Description / 项目简介

基于 bun + Next.js + Move 合约的单体应用，演示 Sui Agent 的受限执行、可审计活动流与一键冻结能力。已接入真实 Google zkLogin + Sui testnet 基础链路，支持稳定 user salt 派生、testnet 余额读取、faucet 请求和探针交易签名上链。Move 合约实现 create / execute_task / freeze 链上路径。
